# AI Deepfake detection

AIfactory와 국과수에서 진행하는 AI deepfake detection 경진대회. 사용자의 얼굴을 중점적으로한 이미지(jpg, png)와 동영상(mp4)이 주어졌을 때 AI로 생성된 Deepfake인지 예측하는 모델을 개발하는 것이 목적이다.

- 학습과 테스트 데이터는 제공되지 않음(비공개). 데이터셋에는 전통적인 얼굴 합성 방식부터 립싱크, AI 생성 이미지/영상이 포함된다.
- 단일 가중치를 가지는 모델이여야 한다.(앙상블 금지)
- 모델 평가는 macro F1‑score으로 진행된다.
- 얼굴에 대한 딥페이크에 한정한다. 간단한 잡티제거나 색조 조정은 딥페이크로 보지 않는다.
- 딥페이크 이진 분류 사람이 구분 할때의 팁
  - 실제 음성과 비언어적 특성의 불일치
  - 눈의 깜빡임 빈도가 너무 일정하거나 너무 많이 혹은 적게 깜빡일 때
  - 피부가 너무 매끈함
  - 잔머리와 배경이 흐릿하게 구분됨
